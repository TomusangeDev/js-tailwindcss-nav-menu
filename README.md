# Responsive Navigation Menu - Tailwind CSS + JavaScript

## Description
This is a **responsive navigation menu** built using **Tailwind CSS**, **a little Custom CSS** and **JavaScript**.  
It features:  

- Desktop navigation links displayed horizontally  
- Mobile hamburger menu with toggle functionality  
- Hover effects on links  
- Fully responsive design for different screen sizes  

This project is perfect for **beginners learning Tailwind CSS and basic JavaScript**.
